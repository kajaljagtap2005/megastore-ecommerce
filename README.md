# 🛒 Megastore: Full-Stack E-Commerce Platform

Megastore is a responsive, full-stack e-commerce web application designed to provide a seamless shopping experience. Built from the ground up, this project features a dynamic customer frontend and a secure, data-driven Owner Dashboard for store management. 

This project demonstrates end-to-end full-stack architecture, including RESTful API design, relational database management, and modern, state-driven UI development.

## ✨ Key Features

### For Customers
* **Modern UI/UX:** Smooth page transitions and interactive elements powered by Framer Motion.
* **Dynamic Cart System:** Users can add/remove items and calculate real-time totals.
* **Secure Authentication:** Complete user registration and login flows.
* **Full Checkout Pipeline:** Captures shipping addresses, contact details, and processes mock transactions.
* **Fully Responsive:** Optimized for flawless viewing across desktop, tablet, and mobile devices.

### For Store Owners (Admin Portal)
* **Role-Based Access Control:** Secure, password-protected route accessible only to administrators.
* **Live Analytics Dashboard:** Tracks real-time revenue, total active orders, and product inventory.
* **Inventory Management:** Form-driven interface to instantly publish new products to the live database.
* **Customer & Order Tracking:** Data tables displaying registered users and comprehensive order histories (including shipping addresses).

## 🛠️ Tech Stack

**Frontend:**
* React.js (Hooks, State Management)
* React Router DOM (Client-side routing & protected routes)
* Framer Motion (UI Animations)
* CSS3 (Flexbox/Grid, Media Queries for Mobile Responsiveness)

**Backend:**
* Node.js & Express.js (REST API creation)
* Sequelize ORM (Database interaction and modeling)

**Database:**
* SQLite (Relational data mapping for Users, Products, and Orders)
